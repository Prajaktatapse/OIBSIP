# Iris Flower Classification

## Objective
Build a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)

## Result
The models were trained and evaluated using accuracy score, confusion matrix, and classification report.
