# Sales Prediction Using Python

## Objective
Build a machine learning model to predict product sales based on advertising expenditure on TV, Radio, and Newspaper.

## Dataset
- Advertising Dataset
- Total Records: 200
- Features:
  - TV
  - Radio
  - Newspaper
- Target:
  - Sales

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Linear Regression
- Random Forest Regressor

## Project Workflow
1. Import libraries
2. Load the dataset
3. Data preprocessing
4. Exploratory Data Analysis (EDA)
5. Data visualization
6. Train-test split
7. Model training
8. Model evaluation
9. Compare model performance
10. Predict sales

## Results
- Linear Regression R² Score: 0.8994
- Random Forest R² Score: 0.9813

Random Forest Regressor performed better than Linear Regression for this dataset.

## Conclusion
A machine learning model was successfully developed to predict product sales using advertising expenditure. The Random Forest Regressor achieved the highest accuracy and provided the best prediction performance for this dataset.
